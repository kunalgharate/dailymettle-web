# 🔥 Dailymettle — 75 Hard Challenge Tracker

The free, offline-first companion app for the **75 Hard Challenge**. Track all 5 daily tasks, build streaks, and stay disciplined for 75 days.

🌐 **Website:** [dailymettle.web.app](https://dailymettle.web.app)
📱 **Download:** [Google Play Store](https://play.google.com/store/apps/details?id=com.seventyfive.hard.challenge)

## ✨ Features

- ✅ **Offline Mode** — Track progress anywhere, no internet needed
- 🔔 **Smart Reminders** — Customizable daily alerts for each task
- 💡 **Daily Motivation** — Curated quotes every morning
- 📊 **Progress Tracking** — Visual streaks and completion stats
- 🎯 **Task Management** — All 5 daily tasks in one clean interface
- ⚡ **Minimal & Fast** — Lightweight, no bloat, loads instantly

## 📋 What is 75 Hard?

75 Hard is a mental toughness challenge that requires completing 5 tasks every day for 75 consecutive days:

1. Two 45-minute workouts (one must be outdoors)
2. Follow a diet with no cheat meals
3. Drink 1 gallon of water
4. Read 10 pages of a non-fiction book
5. Take a progress photo

Miss a single task? Start over from Day 1.
